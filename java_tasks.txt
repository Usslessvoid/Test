Задача #1

Написать программу, которая способна вычислить результат математического выражения представленного строкой.
Выражение может содержать в себе операции сложения, вычитания, умножения и деления. Программа должна учитывать
порядок действий в выражении (умножение и деление выполняются перед сложением и вычитанием). Числа могут быть
дробными.

Входные данные:
  1. Выражение - строка. Например "1.56 + 2 / 3.24 + 4 * 3.2"
Выходные данные:
  1. Результат вычислений - дробное число.
 
 

Задача #2

Необходимо создать echo server, который принимает соединения на порту 8020, и дублирует данные из входящего
потока в исходящий поток. Сервер должен быть способен работать с несколькими соединениями
одновременно и иметь в основе своей реализации сокеты.

Входные данные:
  1. Произвольная UTF-8 строка
Выходные данные:
  1. Копия входных данных

 

Задача #3

Необходимо реализовать возможность хранения информации о книгах и их авторах в базе данных, а также
написать программу, которая может предоставлять информацию о книгах, написанных неким автором.
Стоит учесть, что автор за время своей творческой деятельности может написать несколько книг, а
книга может быть написана несколькими авторами. Можно выбрать любую реляционную базу данных.
Для работы с БД на уровне программы необходимо использовать JDBC. Необходимо будет предоставить не
только исходный код программы, но и SQL скрипт для создания структуры БД и данных.

Входные данные:
  1. Имя автора
Входные данные:
  1. Список книг принадлежащих автору
 
 

 
 
Все задачи обязательны к выполнению. Результаты работы (исходники/скрипты/описания) нужно загрузить в публичный репозиторий на GitHub.
